Element OppositeParallelLinesFeatureDefinitionType / EndRadius1
Annotations
 
The optional EndRadius1 element is the radius of a complex end in the direction opposite the centerline vector. If the end type is flat, then the opposite parallel lines feature has a filleted flat end, and the value must be small enough that a portion of the end is flat. If the end type is round, this value must be such that the end is an outward circular arc. If the end radius is greater than that of a tangent round end then whether the end expands in size like a dumbbell or not is defined by the Expanded element.
