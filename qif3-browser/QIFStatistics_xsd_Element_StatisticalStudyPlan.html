The global StatisticalStudyPlan element is the head of a
substitution group. Any element that is a member of the group may
be substituted for the StatisticalStudyPlan element anywhere the
StatisticalStudyPlan element may be used in an instance file.
This substitution group includes the following members:
  BiasStudyPlan
