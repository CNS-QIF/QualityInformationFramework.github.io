Element PatternFeatureCircularArcNominalType / FirstFeatureLocation
Annotations
 
The FirstFeatureLocation element identifies the nominal feature whose location is the location of the first element of the pattern. The distance from the center of the arc to this feature must equal the radius of the arc. The other nominal features in the feature set must be in locations defined by the pattern.

Complex Type PatternFeatureCircularArcNominalType
Annotations
 
The PatternFeatureCircularArcNominalType defines the pattern feature nominal information for an individual circular arc pattern feature. The nominal location of first element of the circular arc pattern is the nominal location of the first feature in the pattern (which is not necessarily the first feature in the feature set).
