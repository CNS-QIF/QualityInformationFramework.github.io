Simple Type ThreadClassEnumType
Annotations
 
The ThreadClassEnumType enumerates values that describe a standard thread class or a thread tolerance class for the thread fit. Thread tolerance class enumerations are designated in ASME B1.1-2003, Section 4.1 and ISO 965-1. ISO external thread tolerance classes such as "3e" are implemented as "EXT_3E" in QIF.
